# الواجهة الأمامية (Frontend)

## نظرة عامة
الواجهة الأمامية مبنية باستخدام:
- React.js
- TypeScript
- Material-UI للتصميم

## هيكلة المشروع
```
frontend/
├── src/
│   ├── components/    # المكونات القابلة لإعادة الاستخدام
│   ├── pages/        # صفحات التطبيق
│   ├── hooks/        # React Hooks المخصصة
│   ├── services/     # خدمات API
│   └── utils/        # أدوات مساعدة
├── public/          # الملفات العامة
└── styles/         # ملفات التنسيق
```

## الميزات الرئيسية
1. واجهة مستخدم تفاعلية
2. إدارة الحالة باستخدام Redux
3. التوثيق والمصادقة
4. دعم متعدد اللغات

## تشغيل التطبيق
1. تثبيت التبعيات:
```bash
npm install
```

2. تشغيل بيئة التطوير:
```bash
npm start
```

3. بناء نسخة الإنتاج:
```bash
npm run build
```
